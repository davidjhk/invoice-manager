# Invoice Manager 사용자 메뉴얼 (2025년 8월 버전)

이 메뉴얼은 인보이스 매니저 일반 사용자를 위한 안내서입니다.  
인보이스 매니저를 통해 인보이스(청구서), 견적서, 고객, 제품, 회사 정보를 쉽고 안전하게 관리할 수 있으며, AI Helper 기능과 다양한 PDF 템플릿으로 전문적인 문서를 쉽게 작성할 수 있습니다.

---

## 1. 로그인 및 계정 관리

### 1.1 로그인

- 웹사이트 주소로 접속 후, 사용자명과 비밀번호를 입력하세요.
- 관리자, 일반 사용자, 데모 계정이 제공됩니다.

### 1.2 비밀번호 변경

- 우측 상단의 프로필 아이콘 클릭 → "비밀번호 변경" 메뉴 선택
- 새 비밀번호 입력 후 저장

### 1.3 로그아웃

- 우측 상단의 프로필 아이콘 클릭 → "로그아웃" 선택

---

## 2. 회사 정보 관리

### 2.1 회사 정보 수정

- 메뉴에서 "회사 관리" 클릭
- 회사명, 주소, 연락처, 로고, 언어, 테마(다크/라이트), 컴팩트 모드 설정 가능
- PDF 템플릿 선택 (Classic, Modern, Corporate, Elegant, Creative 중 선택)
- 변경 후 "저장" 버튼 클릭

### 2.2 언어 및 테마 변경

- 상단바에서 언어 아이콘 클릭 → 원하는 언어 선택
- 다크모드/컴팩트모드 토글 버튼으로 즉시 변경 가능

---

## 3. 고객 관리

### 3.1 고객 추가

- 메뉴에서 "고객 관리" 클릭
- "새 고객 추가" 버튼 클릭
- 이름, 연락처, 주소 등 입력 후 저장

### 3.2 고객 정보 수정/삭제

- 고객 목록에서 원하는 고객 선택
- "수정" 또는 "삭제" 버튼 클릭

---

## 4. 제품 및 카테고리 관리

### 4.1 제품 추가

- "제품 관리" 메뉴 클릭
- "새 제품 추가" 버튼 클릭
- 제품명, 가격, 설명, 카테고리 선택 후 저장

### 4.2 제품 카테고리 관리

- "제품 관리" → "카테고리" 탭 클릭
- "새 카테고리" 버튼으로 추가
- 드래그 앤 드롭으로 순서 변경
- 활성/비활성 체크박스 사용

---

## 5. 인보이스(청구서) 관리

### 5.1 인보이스 생성

- "인보이스 관리" 메뉴 클릭
- "새 인보이스" 버튼 클릭
- 고객, 제품, 수량, 가격, 세금 등 입력
- AI Helper 기능을 사용하여 제품 설명 자동 생성 가능
- "저장" 버튼 클릭

### 5.2 인보이스 PDF 다운로드/이메일 발송

- 인보이스 상세 화면에서 "PDF 다운로드" 또는 "이메일 발송" 버튼 클릭
- PDF는 회사 설정에서 선택한 템플릿으로 자동 생성됩니다 (Classic, Modern, Corporate, Elegant, Creative)
- 이메일 발송 시, 수신자 이메일 입력 후 전송

### 5.3 인보이스 상태 관리

- 인보이스 목록에서 상태(초안, 발송, 결제 완료 등) 확인 및 변경

### 5.4 AI Helper 사용하기

- 인보이스 항목 작성 시, 제품 설명란 우측의 "AI Helper" 아이콘 클릭
- AI가 자동으로 전문적인 설명을 생성해줍니다
- 생성된 설명 중 원하는 것을 선택하거나 직접 수정 가능
- 결제 조건과 가격 제안도 자동으로 제공됩니다

---

## 6. 견적서 관리

### 6.1 견적서 생성

- "견적서 관리" 메뉴 클릭
- "새 견적서" 버튼 클릭
- 고객, 제품, 가격 등 입력 후 저장

### 6.2 견적서 PDF 다운로드/이메일 발송

- 견적서 상세 화면에서 "PDF 다운로드" 또는 "이메일 발송" 버튼 클릭
- PDF는 회사 설정에서 선택한 템플릿으로 자동 생성됩니다

### 6.3 견적서를 인보이스로 변환

- 견적서 상세 화면에서 "인보이스로 변환" 버튼 클릭

---

## 7. AI Helper 기능

### 7.1 인보이스 항목 설명 자동 생성

- 인보이스나 견적서 작성 시, 제품/서비스명만으로 전문적인 설명 자동 생성
- 항목별로 AI Helper 아이콘 클릭 → 설명 제안 확인 → "설명에 추가"를 클릭 → 필요에 따라 수정

### 7.2 결제 조건 및 가격 제안

- 고객 유형과 금액에 맞는 적절한 결제 조건 자동 제안
- 시장 데이터 기반의 제품/서비스 가격 제안 제공

### 7.3 작업 범위 생성

- 서비스 키워드 입력 시, 전문적인 작업 범위 및 세부 항목 자동 생성
- 기존 고객 정보를 기반으로 맞춤형 제안 제공

---

## 8. US Sales Tax(미국 판매세) 자동 계산

### 8.1 ZIP 코드 입력으로 자동 세율 적용

- 인보이스/견적서 작성 시, 고객 주소의 ZIP 코드가 입력되어 있어야 합니다.
- 자동으로 해당 지역의 판매세율이 적용됨

### 8.2 세율 확인

- 인보이스/견적서 화면에서 세율 정보 확인 가능

---

## 9. 구독 및 플랜 관리

### 9.1 플랜 확인

- 우측 상단 프로필 메뉴에서 현재 구독 플랜 확인 가능
- Free, Standard, Pro 플랜별 기능 제한 안내

### 9.2 플랜 업그레이드

- 더 많은 기능이 필요한 경우 관리자에게 플랜 업그레이드 요청
- 멀티 유저, 멀티 컴퍼니 기능과 템플릿 선택 기능은 Pro 플랜에서만 사용 가능

---

## 10. 기타 기능

### 10.1 다국어 지원

- 상단 언어 아이콘 클릭 → 원하는 언어로 즉시 변경
- 한국어, 영어, 스페인어, 중국어(간체/번체) 지원

### 10.2 다크모드/컴팩트모드

- 상단 토글 버튼 클릭 → 즉시 모드 변경

### 10.3 도움말 및 문의

- 우측 상단 "도움말" 메뉴에서 FAQ 및 문의 방법 확인

---

## 11. 자주 묻는 질문(FAQ)

**Q. PDF에 회사 푸터가 안 나와요.**

- 회사 설정에서 "푸터 숨기기"가 활성화되어 있으면 PDF 푸터가 표시되지 않습니다.

**Q. 이메일이 발송되지 않아요.**

- 이메일은 SMTP2GO 서비스를 사용하므로 해당 서비스의 API를 회사 설정 화면에서 등록해야 합니다.
- SMTP2GO API 등록에 어려움이 있다면 관리자에게 문의해 주세요.

**Q. 세율이 자동으로 적용되지 않아요.**

- 고객 주소의 ZIP 코드가 정확한지 확인하세요.
- 인보이스/견적서 작성시 세금 자동 계산이 선택되어 있어야 합니다.

**Q. 언어/테마가 변경되지 않아요.**

- 브라우저 캐시를 지우거나 다시 로그인해보세요.

**Q. AI Helper 기능이 보이지 않아요.**

- AI Helper는 유료 플랜(Standard, Pro)에서만 사용 가능합니다. 플랜 업그레이드를 하시면 사용이 가능합니다.

**Q. AI Helper가 생성한 설명이 마음에 안 들어요.**

- AI가 생성한 설명은 "설명에 추가"를 클릭하신 후 직접 수정할 수 있으며, 전체적으로 마음에 들지 않으면 다시 생성하실 수 있습니다.

**Q. PDF 템플릿을 바꾸고 싶어요.**

- Pro 버전 이상의 사용자는 회사 설정에서 PDF 템플릿을 변경할 수 있습니다. Classic, Modern, Corporate, Elegant, Creative 중 선택 가능합니다.

**Q. 화면 상으로는 문제가 없는데 PDF 문서에서 한글이 제대로 표시되지 않아요 .**

- 일부 템플릿은 한글을 지원하지 않는 폰트를 사용하므로 한글이 꺠져 나올 수 있습니다. 이 경우에는 회사 설정에서 "PDF에 CJK 폰트 사용" 옵션을 켜시면 정상적으로 표시될 것입니다.
- 중국어가 포함된 문서의 경우에는 위 옵션을 켜고 언어를 언어 설정을 중국어로 선택하신 후 PDF를 다운로드하시면 정상적으로 표시됩니다.
